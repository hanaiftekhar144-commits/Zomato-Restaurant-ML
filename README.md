# Zomato Restaurant Data Analysis & Machine Learning Project

## 📊 Overview
This project focuses on performing Exploratory Data Analysis (EDA) and applying Machine Learning techniques on the Zomato restaurant dataset to gain insights into restaurant trends, customer preferences, and rating patterns.

The dataset includes information about restaurants such as location, cuisines, cost, ratings, and services offered. The analysis helps in understanding factors affecting restaurant ratings and predicting outcomes using machine learning models.

---

## 🎯 Business Objective
The main objective of this project is to:

- Analyze restaurant trends and customer preferences  
- Identify key factors influencing restaurant ratings  
- Understand the impact of cost, location, and services on ratings  
- Build machine learning models for predictive analysis  
- Provide data-driven insights for better decision-making in the food industry  

---

## 🔍 Steps Performed

### 🧹 Data Preprocessing
- Imported dataset and explored structure  
- Handled missing values and inconsistencies  
- Removed duplicates  
- Converted categorical variables using encoding techniques  
- Feature engineering:
  - Extracted relevant features  
  - Processed location and cuisine data  
- Prepared dataset for analysis and modeling  

---

## 📁 Dataset Overview

- Dataset contains information about:
  - Restaurant Name  
  - Location  
  - Cuisines  
  - Average Cost for Two  
  - Ratings  
  - Votes  
  - Online Delivery / Table Booking  

---

## 📈 Exploratory Data Analysis (EDA)

- Distribution of restaurant ratings  
- Popular cuisines analysis  
- Location-wise restaurant distribution  
- Cost vs Rating analysis  
- Votes vs Rating relationship  
- Online delivery and table booking impact  
- Identified top-rated restaurants and cities  

---

## 🤖 Machine Learning Implementation

The following supervised learning models were implemented:

- Logistic Regression  
- Random Forest Classifier  
- XGBoost Classifier  

### ⚙️ ML Workflow
- Data splitting (Train/Test)  
- Feature selection  
- Model training and evaluation  
- Performance comparison  

---

## 📊 Key Insights

- Restaurants with higher votes tend to have better ratings  
- Certain cuisines are more popular and highly rated  
- Cost influences customer perception and ratings  
- Online delivery availability impacts restaurant popularity  
- Location plays a major role in restaurant success  

---

## 💡 Recommendations

- Focus on high-demand cuisines in popular locations  
- Improve customer engagement to increase votes and ratings  
- Optimize pricing strategies based on customer preferences  
- Promote online delivery services for better reach  
- Use data-driven strategies for restaurant expansion
